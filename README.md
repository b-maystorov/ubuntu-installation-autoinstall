# Ubuntu Installation / Autoinstall

Dieses Repository dokumentiert ein Lern- und Praxisprojekt zur Installation und Grundkonfiguration von Ubuntu-Systemen.

Der Fokus liegt auf einem sauberen, nachvollziehbaren Installationsprozess, technischen Entscheidungen und FISI-relevanter Dokumentation.

---

## Ziel des Projekts

Ziel ist es, eine Ubuntu-Installation so zu planen, durchzuführen und zu dokumentieren, dass sie für Schulungs-, Test- oder Laborumgebungen nachvollziehbar wiederholt werden kann.

Dabei werden typische Aufgaben aus der Systemintegration geübt:

- Installation eines Linux-Systems
- Grundkonfiguration nach der Installation
- Benutzer- und Rechteverwaltung
- Netzwerkgrundlagen
- Dokumentation technischer Entscheidungen
- Vorbereitung für spätere Automatisierung

---

## Verwendete Technologien

| Bereich | Technologie |
|---|---|
| Betriebssystem | Ubuntu / Ubuntu Server |
| Installation | ISO-Installation, Autoinstall-Grundlagen |
| Konfiguration | Shell, Bash, Systemtools |
| Sicherheit | Benutzerrechte, SSH, Updates |
| Dokumentation | Markdown, Git, GitHub |
| Virtualisierung | QEMU/KVM oder VM-Umgebung |

---

## Projektstand

Aktueller Stand:

- Repository erstellt
- Grundstruktur vorbereitet
- README-Dokumentation begonnen

Geplant:

- Installationsschritte dokumentieren
- Screenshots ergänzen
- Partitionierung erklären
- Benutzer- und SSH-Konfiguration dokumentieren
- mögliche Autoinstall-Konfiguration ergänzen
- Lessons Learned ergänzen

---
